Certificate for Rennie Ramlochan, Google IT Automation with Python Professional Certificate, offered through Google.

Those who earn the Google IT Automation with Python Professional Certificate have completed six courses, developed by Google, that include hands-on, practice-based assessments designed to prepare them for roles in IT support and automation. They are competent in utilizing Python to interact with operating systems, managing system resources, automating tasks, and leveraging Git and GitHub for version control.

The focus of this Portfolio was to gather, improve, and complete activities related to IT automation and presented in Google’s IT Automation with Python Professional Certificate offered through Coursera.

Course Curriculum
1. Crash Course on Python
This course covers the fundamentals of Python programming to get you up and running with the language.

Key Skills: Python syntax, data types, loops, functions, and basic object-oriented programming concepts.

Hands-on: Writing scripts to automate simple tasks and performing basic data manipulation.

2. Using Python to Interact with the Operating System
Learn to manipulate files and processes on your computer using Python scripts.

Key Skills: Navigating the file system, managing files and directories, reading/writing files, and working with system processes.

Hands-on: Creating scripts that organize files, process system information, and manage system resources.

3. Introduction to Git and GitHub
Master the essential version control system used by developers globally to track code changes and collaborate.

Key Skills: Git basics (commit, push, pull), branching, merging, and managing remote repositories on GitHub.

Hands-on: Setting up repositories, managing pull requests, and resolving merge conflicts.

4. Troubleshooting and Debugging Techniques
Learn how to identify, analyze, and fix software problems efficiently.

Key Skills: Debugging strategies, identifying root causes of issues, using logging, and binary search techniques.

Hands-on: Fixing bugs in existing codebases and writing tests to prevent regression.

5. Configuration Management and the Cloud
Explore how to manage computers at scale using configuration management tools and cloud services.

Key Skills: Infrastructure as Code (IaC), Puppet basics, interacting with cloud APIs, and managing remote instances.

Hands-on: Automating system configuration and deploying services to the cloud.

6. Automating Real-World Tasks with Python
The capstone course where you apply everything you’ve learned to solve authentic IT problems.

Key Skills: Processing log files, interacting with web services via APIs, generating reports, and sending automated emails.

Hands-on: Building an end-to-end automation project that integrates Python, web APIs, and system management.

Directory Structure
The repository follows a directory structure that separates portfolio activities by course module:

Course 01 Portfolio Activities: Solutions and documentation for assignments in the Crash Course on Python.

Course 02 Portfolio Activities: Solutions and documentation for assignments regarding Operating System interaction.

(And so on for each course in the program.)

Getting Started
To consult this repository for reference:
Navigate to the relevant course directory for the material you are interested in. Inside each directory, you will find a section labeled "Portfolio Activities." You may consult the solutions and documentation provided for each activity. Please note that the solutions are for reference purposes, and there is no guarantee that they are the best or only solutions.
