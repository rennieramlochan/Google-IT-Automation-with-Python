The focus of this Portfolio was to gather, improve and complete activities related to IT Automation and presented in Google's IT Automation with Python Professional Certificate offered through Coursera.
This repository serves as a professional portfolio documenting the completion of the Google IT Automation with Python Professional Certificate. 

The program consists of six courses designed to equip learners with essential skills for IT support, systems administration, and task automation.
The projects and activities contained herein demonstrate proficiency in Python programming, interacting with operating systems, version control, troubleshooting, configuration management, and cloud infrastructure.

Program Overview
Those who earn the Google IT Automation with Python Professional Certificate have demonstrated competency in:
Python Programming: Writing scripts to automate system tasks.
Operating Systems: Manipulating files, processes, and system resources.
Version Control: Using Git and GitHub for collaborative software development.
Troubleshooting: Identifying root causes and debugging complex IT issues.
Configuration Management: Managing system configurations at scale.
Cloud Interaction: Interacting with cloud APIs and managing remote instances.

Course Curriculum
1. Crash Course on Python
Foundations of Python programming.
Key Topics: Syntax, data types, loops, functions, and basic OOP.
Activities: Scripting for automation and data manipulation.

2. Using Python to Interact with the Operating System
Bridging the gap between code and system administration.
Key Topics: File system navigation, process management, and system information parsing.
Activities: Automating file organization and system monitoring.

3. Introduction to Git and GitHub
Version control workflows for collaborative development.
Key Topics: Repository management, branching, merging, and pull requests.
Activities: Establishing version-controlled development environments.

4. Troubleshooting and Debugging Techniques
Systematic approaches to software and system maintenance.
Key Topics: Debugging strategies, logging, and root cause analysis.
Activities: Resolving bugs and implementing regression testing.

5. Configuration Management and the Cloud
Scaling infrastructure and managing remote services.
Key Topics: Infrastructure as Code (IaC), Puppet, and Cloud API integration.
Activities: Automating system configuration and remote deployment.

6. Automating Real-World Tasks with Python (Capstone)
Integrated application of course skills to solve authentic IT problems.
Key Topics: API interaction, log processing, and automated reporting.
Activities: Building an end-to-end automation pipeline.
